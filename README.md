
# Obsedian Project

### Backend
1. cd backend
2. Fill `.env` 
3. npm install
4. npm run dev   (or `npm start`)

### Frontend
1. cd frontend
2. npm install
3. npm run dev
4. Open the URL shown by Vite (usually http://localhost:5173)

## Files included
- backend/server.js
- backend/package.json
- backend/.env.example
- frontend/*
- README.md
